# How to use lists

Lists can be used to help users find a specific item and act on it.

When using lists,
* Order list items in logical ways
* Keep items short and easy to scan
* Show icons, text, and actions in a consistent format

# Guidelines

* Start your item labels with a capital letter
* Don't use commas or any punctuation at the end of each item
